# Wormhole or Die

A fast-paced arcade tunnel-flight game for **iPhone, iPad, and Apple Vision Pro**.

A neon tunnel of obstacles rushes toward you — each wall has a gap, so thread it.
The closer you cut it, the more you score; chain close passes to climb a
multiplier. Play through themed stages, chase endless **Ranked** runs and Game
Center leaderboards, or go head-to-head in real-time **cross-play 1v1 versus**
between iPhone and Apple Vision Pro.
